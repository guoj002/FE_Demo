自定义属性/CSS变量

为了和其他属性区分开，命名的前缀必须是两道横杠（--），然后写上你想要的名字

用一个叫作var()的函数去引用自定义属性的值

var()函数支持第二个参数，代表一个默认值。假如一个变量被声明的时候，第一个参数没有被声明，那么第二个参数值就会被引用

如果var()被定义为一个无效值，这个属性会被定义为它的初始值。
